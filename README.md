# timestamp-api
timestamp API for [FreeCodeCamp](https://www.freecodecamp.com/challenges/timestamp-microservice) with node and express

## Dependencies
```
"dependencies": {
  "express": "^4.14.1",
  "nodemon": "^1.11.0",
  "pug": "^2.0.0-beta11",
  "stylus": "^0.54.5"
}
 ```
 
 ## Using
 `node run watch`
 
 or
 ```
 node run build-css
 node run start
 ```
